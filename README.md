# Metalytics
Overview

Metalytics is an end-to-end data pipeline designed to ingest, process, and analyze precious metals trading data. The project focuses on building a structured data workflow using Python, PostgreSQL, and AWS, with support for analytics and machine learning experiments.

🚀 Features
Data ingestion from external sources (Kaggle API)
ETL pipeline for cleaning and transforming raw data
PostgreSQL database design for structured storage
Historical data retention and backup automation
Basic machine learning models for price trend prediction
Dockerized setup for consistent deployment
